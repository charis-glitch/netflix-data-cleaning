# Netflix Titles Data Cleaning & Exploration

## Overview
This project demonstrates a complete data cleaning and basic exploratory analysis workflow on the Netflix Titles dataset (`netflix_titles.csv`).  

It includes:  
- Loading and inspecting raw data  
- Cleaning duplicates, missing values, and whitespace  
- Converting and handling date columns    
- Filtering unrealistic release years  
- Saving a cleaned dataset  
- Basic visualizations (top countries, genres)    

This notebook is designed for **portfolio, freelance, or practice purposes**.
