# Netflix Titles Data Cleaning & Exploration

## Overview
This project showcases a complete **data cleaning and exploratory analysis workflow** on the **Netflix Titles dataset (`netflix_titles.csv`)**.  
It demonstrates practical data preparation and visualization techniques using **Python, Pandas, and Matplotlib** — suitable for portfolio display, freelance projects, or self-practice.  

### Key Steps:
- Load and inspect raw data  
- Handle duplicates, missing values, and inconsistent formatting  
- Clean and convert date columns (`date_added`, `release_year`)  
- Filter unrealistic or invalid entries  
- Save a cleaned dataset for future analysis  
- Create basic visualizations (top countries, top genres, release trends)

## Insights
- The **United States** produces the most Netflix content, followed by **India** and the **United Kingdom**.  
- **Drama** and **Comedy** are the dominant genres, with **Documentaries** and **Action & Adventure** also highly represented.  
- Seasonal patterns are visible — certain months see higher title additions.  
- Missing director and cast data highlight common data quality issues in real-world datasets.
